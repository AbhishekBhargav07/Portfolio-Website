# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built to showcase my technical skills, projects, certifications, resume, and achievements. This portfolio serves as a central place to highlight my journey as a Computer Science student and aspiring Software Developer.

## 🚀 Features

* Fully responsive design for desktop, tablet, and mobile devices
* Clean and modern user interface
* About Me section
* Technical Skills showcase
* Featured Projects with live/demo links
* Resume download
* Certifications & Achievements
* Contact section with social media links
* Smooth scrolling and interactive animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎯 Purpose

The primary goal of this project is to create a professional online presence where recruiters, collaborators, and developers can explore my technical skills, projects, and accomplishments.
